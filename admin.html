4 Le coller sur GitHub

  1. Sur ton dépôt GitHub, ouvre le fichier articles.js et clique sur l'icône crayon (✏️) en haut à droite pour l'éditer.
  2. Trouve la toute première ligne const ARTICLES = [. Place ton curseur juste après le [, à la ligne suivante.
  3. Colle le bloc que tu viens de copier. Ton nouvel article apparaîtra en premier sur le blog. (Pour le mettre ailleurs, colle-le entre deux articles existants, après une accolade },.)
  4. Descends en bas de la page, clique sur Commit changes (le bouton vert), puis confirme.
  5. Attends une à deux minutes : GitHub Pages republie le site tout seul. Recharge ton blog, l'article est là.

Astuce : le bloc se termine déjà par une virgule, donc tu peux en empiler autant que tu veux les uns sous les autres.
